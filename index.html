REST definition

REST - is an abbreviation that stands for “Representational State Transfer”. It represents the architectural style of interaction between components of a distributed application on the network.

REST properties

When a RESTful API is called, the server will transfer to the client a representation of the state of the requested resource.

REST basic components

    2 things that you need to provide to the server:
  • An identifier for the resource you are interested in.
  • The operation you want the server to perform on that resource, in the form of an HTTP method (GET, POST, PUT, and DELETE).
